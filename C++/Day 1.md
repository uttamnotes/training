## **Basic Structure of a C++ Program**

A C++ program is divided into the following main parts:

1. **Preprocessing Statements**

   * Include required header files
   * Example:

     ```cpp
     #include <iostream>
     ```

2. **Main Function**

   * Execution of the program starts from the `main()` function.

   ```cpp
   int main()
   {
       // program body
       return 0;
   }
   ```

3. **Program Body**

   * Contains variable declarations, input/output statements, and logic.

4. **Return Statement**

   * `return 0;` indicates successful execution of the program.

## **Important C++ Statements & Syntax**

### 1. Output Statement (cout)

Used to display output on the screen.

```cpp
cout << "Hello World" << endl;
```

* `<<` : Insertion operator
* `endl` : Moves the cursor to the next line

---

### 2. Variable Declaration

Syntax:

```cpp
datatype variable_name = value;
```

Example:

```cpp

```

### 3. Statement Terminator

* Every C++ statement **must end with a semicolon (`;`)**
* The semicolon acts as a **delimiter**.

Example:

```cpp

```
